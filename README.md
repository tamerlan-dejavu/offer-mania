# Offer-Mania 🎯

AI-powered interview simulator that helps developers prepare for technical interviews.

## Features
- Technical interview simulation (Python, databases, async)
- HR screening mode
- Algorithmic section (Live Coding)
- System Design mode
- Intelligent post-interview report with personalized recommendations

## Tech Stack
**Backend:** Python, FastAPI, PostgreSQL, SQLAlchemy  
**Frontend:** React, Vite, Tailwind CSS  
**AI:** Anthropic Claude API
